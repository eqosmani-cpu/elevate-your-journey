# Project Memory

## Core
MindPitch: Mental coaching app for football players. Light warm theme default.
Primary: #3A5C4A. BG: #F5F4F0. Gold: #B8976A.
DM Serif Display headings ONLY, DM Sans body. German UI, English code.
Apple-style minimal: never bold (max 500), never scale hover, never spinners.
Lovable Cloud enabled. Auth with email/password. Profile auto-created on signup.
